Фильтры для поиска

Пример: ext: pdf, audio: Arctic Monkeys.

  • ext: - поиск по расширениям файлов
  • doc: - поиск по названиям файлов
  • note: - поиск по названиям заметок
  • page: - поиск по названиям страниц
  • audio: - поиск по аудиофайлам
  • video: - поиск по видеозаписям
  • link: - поиск по ссылках
  • poll: - поиск по опросах
  • geo: - поиск по названям гео-позиции

Если фильтр не указан, то поиск осуществляется по всем фильтрам.

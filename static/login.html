Fave Controller
Что это? Fave Controller — приложение для работы с закладками в социальной сети Вконтакте.
Что умеет?
  • Фильтровать записи по типам. Музыка, Фото, Видео, Документы, Заметки, Граффити, Ссылки, Опросы, Места, Страницы, Альбомы
  • Фильтровать записи по источникам. Instagram, Iphone, Android, Windows Phone, VK, Widget, API, RSS, SMS
  • Искать записи по ключевому слову.
  • Фильтровать записи по дате.
Что-то ещё? Приложение кроссплатформенное, доступно для Windows, OS X и Linux.

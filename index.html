Добро пожаловать

Идея родилась ещё пару лет назад, когда я начал сохранять музыку, книги, ссылки, и закладки стали помойкой. Со временем поиск информации сводился к простой прокрутки страницы, но с увеличением количества записей на странице, Google Chrome просто падал. Так было забыто много записей. До этого момента :)

Цель Fave Controller решить эту проблему.

Возможности:

  • Поиск по ключевому слову.
  • Фильтр записей по типам.
    Музыка, Фото, Видео, Документы, Заметки, Граффити, Ссылки, Опросы, Места, Страницы, Альбомы
  • Фильтр записей по источникам.
    Instagram, Iphone, Android, Windows Phone, VK, Widget, API, RSS, SMS
  • Фильтр по дате.
  • Возможность снимать/ставить лайки.

Установка

Перед установкой понадобятся:

  • node.js - для работы приложения.
  • git - для внесения своих правок в проект :)

После установки и скачивания исходников, нужно сделать следующие команды в терминале:

$ npm install electron-prebuilt -g
$ cd some_path_to_project/FaveController
$ npm install --production
$ electron .

При разработке:

$ cd some_path_to_project/FaveController
$ npm install
$ gulp

Для сборки:

$ cd some_path_to_project/FaveController
$ gulp build
